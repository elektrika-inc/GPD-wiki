Introduction

GPD is a program that works with Thymatron ECT devices to provide users with a modern workflow, including real-time trace monitoring, a patient database, treatment data storage and review, clinical EMR interfacing via HL7, analysis of parameters over time, and data export to common formats.

For a more thorough overview of GPD's features, see the online Features page.

Download and Installation

GPD runs on Windows computers from XP onward.

Complete Distribution

The software is distributed as a ZIP file. Simply unpack it wherever you prefer, and run GPD.exe to start the program. It is recommended to run an auto-update (see below) after installing the main distribution, as sometimes the most recent changes are distributed only as an update.

Link:  GPD-Distribution-ZIP

Update

If you have already installed the program, you can update it by running the auto-updater. To access it,

 Help menu →About…→Check for Update

Note: the auto-update system changed in version 5.4.4, so if you're updating from an earlier version, you'll need to update twice – once to get to 5.4.4, and once to go from there to the latest version.

You'll need to enter license information the first time you run the program. Thymatron users can get a license from their distributor at no cost; or for provisional use, you may enter the license information below:

Organization of This Wiki

Rather than documenting the program screen-by-screen, this wiki is organized into guides by the types of tasks a user might perform.

All users should read the Getting Started section to learn how to create a new user, add a patient, and collect treatment data. After that, one may skip to the particular task one wishes to perform.

The documentation for each task also describes the GPD Preferences that affect program operation in that area.

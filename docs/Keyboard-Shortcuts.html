Keyboard Shortcuts

Login Window

Ctrl+G Login as guest user
Ctrl+Alt+P Preferences
Ctrl+Q Quit Program

Patient List

Ctrl+O Open Patient by ID
Ctrl+E View selected patient's treatments
Ctrl+W Logout
Ctrl+Q Quit Program
Ctrl+F Go to Search field
Ctrl+N Collect new treatment for selected patient
Ctrl+U Show only unfinalized patients
Ctrl+Alt+P Preferences

Treatment List

Ctrl+E View treatment graph
Ctrl+W Close Treatment List
Ctrl+F Go to Search field
Ctrl+N Collect new treatment
Ctrl+K Mark treatment as finalized
Ctrl+Shift+K Unmark treatment as finalized
Ctrl+I Edit Patient Info
Ctrl+R Treatments chart
Ctrl+Shift+R Default Treatments chart
Ctrl+Shift+P Printable Treatments chart
Ctrl+G Treatment graph
Ctrl+Alt+P Preferences

Treatment Chart

Enter Edit treatment info
Ctrl+I Edit Patient Info

Ctrl+T Insert template field
Ctrl+L Complete current field
Ctrl+M Insert all missing template fields

Monitoring / Collection Window

Ctrl+W Close window
F5 Start/Stop Collection
